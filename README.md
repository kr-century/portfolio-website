# Keshav Kumar - Personal Portfolio Website

Welcome to the source code of my personal portfolio website. I am a Full Stack Web Developer and AI Automation Enthusiast passionate about building production-ready, highly optimized web applications.

This project is designed to be a high-performance, multi-page portfolio that strictly follows modern web standards, semantic structuring, and WCAG accessibility guidelines.

## 🚀 Key Features

* **100+ Lighthouse Score:** Highly optimized for performance, accessibility, best practices, and SEO.
* **WCAG Accessibility:** Implements ARIA labels, semantic HTML5 tags (`<header>`, `<nav>`, `<main>`, `<article>`), and tab-navigable UI components for screen readers.
* **Responsive Design:** Mobile-first approach utilizing CSS Grid, Flexbox, and Media Queries for seamless viewing across all devices.
* **Modern UI/UX:** Features a clean, minimalistic dark theme using CSS custom properties (variables) for consistent styling and easy theming.
* **Accessible Contact Form:** Fully keyboard-navigable form with properly linked `<label>` and `<input>` tags.

## 💻 Tech Stack

* **Markup:** HTML5 (Semantic Structure)
* **Styling:** CSS3 (Variables, Grid, Flexbox, Accessible Focus Rings)
* **Scripting:** Vanilla JavaScript (ES6+) for DOM manipulation and accessible mobile navigation.

## 📂 File Structure

* `index.html`: Hero section and introduction.
* `about.html`: Academic background, skills, certifications, and internship details.
* `projects.html`: Showcase of real-world projects including AI-assisted development and full-stack repositories.
* `contact.html`: Accessible communication form.
* `style.css`: Global stylesheet.
* `script.js`: Mobile menu and ARIA attribute management.
